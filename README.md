# Jvim

My vim config file.

## Install

1. mkdir ~/jvim

2. wget https://raw.githubusercontent.com/jiemoon/jvim/master/vimrc ~/jvim/

3. ln -s ~/jvim/vimrc ~/.vimrc

4. ln -s ~/jvim ~/.vim

5. install vundle

    git clone https://github.com/gmarik/Vundle.vim.git ~/.vim/bundle/Vundle.vim

6. open vim, and input `:BundleInstall`
